# Greenfield-Project

## 環境構築

### installing

- フロントエンドにパッケージを追加
```.sh
npm install react -w frontend 
```

- バックエンドにパッケージを追加
```.sh
npm install express -w backend 
```

### 実行方法
※ 一番上の階層から実行できます
- フロントエンドの実行
```.sh
npm run dev -w frontend 
```

- バックエンドの実行
```.sh
npm run dev -w backend 
```
